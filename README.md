# Sandbox

This project is for me to prototype new things.

# Getting Started

1. git clone
1. Install npm modules and bower modules at the root level

    ```
npm install
bower install

    ```

1. Start the database

    ```
npm run gulp
    ```

1. Start the database

    ```
mongod
    ```

1. Start the server

    ```
npm start
    ```


1. Start the server

    ```
npm run gulp watch
    ```
